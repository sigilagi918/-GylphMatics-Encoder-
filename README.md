# -GylphMatics-Encoder-
GlyphMatics: Browser-based steganographic file transport. Any file/folder → TAR → zlib → Braille Unicode (U+2800-U+28FF). Canvas glyph emulator, pattern analysis, Python execution via Pyodide. Move binaries through text channels—email, chat, QR codes. Single HTML, zero backend. v1.7 production-ready.
